# 运行和部署您的 AI Studio 应用

此文件包含您在本地运行应用所需的所有内容。

在 AI Studio 中查看您的应用: https://ai.studio/apps/drive/1Gs74BvjhCkITxgA2sMUZDRdZ4DHIi6HQ

## 本地运行

**先决条件:** Node.js


1. 安装依赖:
   `npm install`
2. 在 [.env.local](.env.local) 中设置 `GEMINI_API_KEY` 为您的 Gemini API 密钥
3. 运行应用:
   `npm run dev`
